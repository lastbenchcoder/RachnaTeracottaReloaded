STORES

Below api's will help you to Create, Update, List, Detail the stores

CREATE NEW STORE:

Below URL is used to create the new store, it is POST method requires json input.

URL: http://www.all4road.com/api/stores

TYPE : POST

Content-Type: application/json

INPUT JSON: 

            {
                "Store_Name":"Rachna TeracottaStore",
                "Store_Description":"Online Rachna Teracotta Testing Store",
                "Store_Logo":"http://rachnateracotta.com/content/logo.png",
                "Store_Status":"Active"
            }

 

LIST STORE:

Below URL is used to list all the created stores, it is GET method does not require any input.

URL: http://www.all4road.com/api/stores

 

STORE DETAIL:

Below URL is used to get the detail of the stores, it is GET method does not require storeid as a parameter.

URL: http://www.all4road.com/api/stores/{storeid}

 

UPDATE STORE:

Below URL is used to update the new store, it is POST method requires json input.

URL: http://www.all4road.com/api/stores/{store id to update}

TYPE : POST

Content-Type: application/json

INPUT JSON: 

           {
                "Store_Id":1,
                "StoreCode":"RT1STRECODE2",
                "Store_Name":"Rachna TeracottaStore Updated",
                "Store_Description":"Online Rachna Teracotta Testing Store",
                "Store_Url":"?storecode=RT1STRECODE2&redirect-url=Rachna TeracottaStore.html",
                "Store_Logo":"http://rachnateracotta.com/content/logo.png",
                "Store_CreatedDate":"2019-07-07T02:58:18.993",
                "Store_UpdatedDate":"2019-07-07T02:58:18.993",
                "Store_Status":"Active"
            }

INVITATIONS

Below api's will help you to Create, Update, List, Detail the Invitations

CREATE NEW INVITATION:

Below URL is used to create the new invitations, it is POST method requires json input.

URL: http://www.all4road.com/api/invitations

TYPE : POST

Content-Type: application/json

INPUT JSON: 

           {
                "Role":"SuperAdmin",
                "Store_Id":1,
                "Invitation_EmailId":"admin@rachnateracotta.com",
                "Invitation_Status":"Active",
                "Send_Activity_Mail":1
            }

 

LIST INVITATIONS:

Below URL is used to list all the created invitations, it is GET method does not require any input.

URL: http://www.all4road.com/api/invitations

 

INVITATIONS DETAIL:

Below URL is used to get the detail of the invitation, it is GET method does not require storeid as a parameter.

URL: http://www.all4road.com/api/invitations/{invitationid}

 

UPDATE INVITATIONS:

Below URL is used to update the invitation, it is POST method requires json input.

URL: http://www.all4road.com/api/invitations/{invitation id to update}

TYPE : POST

Content-Type: application/json

INPUT JSON: 

          {
            "Invitation_Id":1,
            "Invitation_Code":"RT1INVTCODE2",
            "Role":"SuperAdmin",
            "Store_Id":1,
            "Invitation_EmailId":"admin@rachnateracotta.com",
            "Invitation_Status":"Active",
            "Invitation_CreatedDate":"2019-07-07T06:06:14.033",
            "Invitation_UpdatedDate":"2019-07-07T06:06:14.033",
            "Send_Activity_Mail":1
        }
